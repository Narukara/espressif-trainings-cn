# 入门教程

入门教程包含基础的嵌入式开发教学。
在此教程的结尾，我们将能够与外界环境交互，包括与板上的传感器通讯。入门教程的内容包括：
- 项目概览
- 用 cargo generate 生成一个项目。
- 编写一个 HTTP 客户端。
- 编写一个 HTTP 服务器。
- 编写一个 MQTT 客户端，它能够：
  - 发布传感器数据
  - 通过订阅的 topic 接收命令。
